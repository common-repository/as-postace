Replaces the default text editor for posts / pages with ACE.

== Description ==

This plugin replaces the default text editor (HTML Tab) on posts & pages with
a code highlighting, fancy text editor called ACE.

== Installation ==

1. Upload `/as-postace/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy.
